# Neutrino
## Installation
### Yarn
```bash
$ yarn add https://github.com/nuclearide/neutrino
```
### NPM
```bash
$ npm install https://github.com/nuclearide/neutrino
```
## Running
```bash
$ npx neutrino ./path/to/main/file.js
```
